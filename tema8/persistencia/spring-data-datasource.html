Spring data

Datasource

Representa la conexión (URL, credenciales, etc.) a una base de datos específica usando su driver JDBC. A continuación, vamos a crear un proyecto Spring Data JPA y configurar su base de datos1.

  1. Genera el esqueleto del proyecto y ábrelo en IntelliJ.

  2. Añade la siguiente configuración al fichero de propiedades de la aplicación Spring Boot:

    spring.datasource.url=jdbc:h2:file:./dbdata
spring.datasource.username=sa
spring.h2.console.enabled=true
spring.jpa.generate-ddl=true
spring.sql.init.mode=always
spring.sql.init.continue-on-error=true
spring.jpa.defer-datasource-initialization=true

  3. Arranca la aplicación Spring Boot, y conéctate a la consola de administración de la BD utilizando las propiedades del datasource anteriores.

1
