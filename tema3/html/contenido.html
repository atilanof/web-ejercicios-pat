La etiqueta <body> contiene los elementos visibles. Existen diferentes tipos, aquí vamos a ver algunos relacionados con bloques de texto.

Tablas

Crea una tabla usando <table>, <tr> para las filas, y <td> para las celdas, con el siguiente contenido. Luego separa la cabecera con <thead> del resto con <tbody>.

| Tipo       | Elementos     |
|------------|---------------|
| Título     | h1, ... , h6  |
| Párrafo    | p             |
| Listas     | ul, ol, li    |
| Tablas     | table, tr, td |
| Separador  | hr            |

Caracteres reservados

Hay otras etiquetas que solo deben ir dentro de bloques de texto. Un ejemplo es <br> que permite añadir un retorno de carro1. Además, los textos no pueden contener directamente los caracteres < y > porque podrían interpretarse como código HTML. Para usarlos se han de escribir como entidades HTML:

&lt;
&gt;
<br>
&amp;
&quot;
1

HTML ignora los retornos de carro normales introducidos en código con la tecla ↲.
